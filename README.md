Original README.txt provided for the graders:

PACKAGES:
Given the test environment, there are no additional required packages
USAGE:
To score our model, please open the "train_and_test_final.ipynb"
To import the training and testing data, please find at the top of the
"train_and_test_final.ipynb" the huge block comment:
# -=-=-=-=-=-=-=-=-=-=-=--==-
# README: Define the csv locations
# -=-=-=-=-=-=-=-=-=-=-=--==-
Then below the comment, there are file import lines
The file paths for all_labels_np and all_images_np should correspond to the
training and validation labels
The file paths for x_test_easy_np and t_test_easy_np should correspond to the test
images and truth labels for the Easy test set
The file paths for x_test_hard_np and t_test_hard_np should correspond to the test
images and truth labels for the Hard test set
ADDITIONAL NOTES:
"experiments.ipynb" is code for all of our experiments and should not be used for
scoring.
